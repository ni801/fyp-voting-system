 Blockchain-Based Secure Online Voting System

Final Year Project  
The Islamia University of Bahawalpur 
Department of IT

A decentralized web application for secure, transparent, and tamper-proof online voting using Ethereum blockchain and Solidity smart contracts.

 Features
- Voter registration & MetaMask wallet login
- One-person-one-vote enforcement
- Immutable vote storage
- Real-time results dashboard
- Admin panel to manage election

 Tech Stack
- Solidity (Smart Contracts)
- React.js + Web3.js/Ethers.js
- Hardhat + Ganache
- MetaMask
